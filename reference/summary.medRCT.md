# Summarise Results from Mediation Analysis

Provides a summary of the results obtained from a mediation analysis
conducted using the `medRCT` function.

## Usage

``` r
# S3 method for class 'medRCT'
summary(object, ...)
```

## Arguments

- object:

  An object of class `medRCT`.

- ...:

  Additional arguments to customise the summary output.
